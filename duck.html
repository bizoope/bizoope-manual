# データダウンロード

このメニューでは、POSデータのテキストダウンロードできます。任意の期間と集計単位、任意の店舗、商品を指定してのダウンロードが可能です。

# リスト画面

リスト画面について説明します。

リスト画面はリストとステータスで構成されています。

# リスト

リストについて説明します。

① データを作成

クリックすると分析条件画面に飛びます。

② 更新

ページ(リスト)の更新を行います。

※ページ(リスト)は自動で更新されないので随時更新を行ってください。

③ 削除

リスト内のデータを選択し、削除ボタンをクリックすることでそのデータをリストから削除できます。

④ アイコン

アイコンは4種類あります。

# ステータス

リストからデータを選択すると分析条件が表示されます。

①ダウンロード

データをダウンロードします。

②条件を再利用

分析条件に飛び、選択したデータの分析条件を復元します。

# 分析条件画面

分析条件画面の説明をします。

この画面では、分析条件を指定します。

条件設定画面の共通操作はこちら

# 基本条件

基本条件について説明します。

  • 説明

入力した文がリストの説明欄に表示されます。

  • 追加ダウンロード

選択することで、商品マスタ・店舗マスタのファイルを併せてダウンロードできます。

詳細はダウンロードファイルへ

# 商品分類

※商品分類で選択できる商品・分類は契約時の分類のみです。

# オプション

オプションではダウンロードファイルの文字コードを指定できます。

# ダウンロードファイル

ファイルはzipファイルにまとめてダウンロードします。

区切り文字はカンマ(,)、文字コードはオプションで指定したものです。

# 商品マスタ

追加ダウンロードで商品マスタを選択すると、items.csv/ categories.csvのファイルを併せてダウンロードできます。

items.csvは、商品コード・商品名・分類コード・メーカーコードをまとめたデータです。

categories.csvは、カテゴリーコードをまとめたデータです。

# 店舗マスタ

追加ダウンロードで店舗マスタを選択すると、stores.csvのファイルを併せてダウンロードできます。

stores.csvは、店舗コード・店舗名・店舗の郵便番号をまとめたデータです。
