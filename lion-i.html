# リピート推移

このメニューでは、1列目の購入顧客が、その後どのようになって行ったのかを商品ごとに推移表示で示します。

リピートを狙った商品販促の検証等に利用します。

# 条件設定画面

条件設定画面の共通操作はこちら

リピート推移の条件設定の特徴的な部分について説明をします。

# 期間

設定する期間、期間単位によって表示が変わります。

# 期間単位の設定

期間単位とは、複数期間を１つの列にまとめて出力する際の単位です。

①：期間の単位（週／月）の変更ができます。

②：設定した期間単位ごとに表示されます。

# 出力する期間の設定

③：出力したい期間の始まりを設定できます。

④：選択した期間の単位（週/月）に従って表示する期間の範囲を設定できます。

上記の画像を例にすると

11月2日から1週間ごとに13週間分表示されます。

# オプション(初購入)

初購入に関する設定ができます。

ある商品の購入者のうち、③から指定日数過去に遡って購入履歴のない事を「初購入」と定義します。

該当する顧客は出力結果画面の1列目に初購入として表示されます。

# リピート推移の軸

# 1列目（基準の期間）

軸説明
購入者＝購入
購入＝再購入＋初購入
再購入③から指定日数過去に遡って購入履歴のある顧客
初購入③から指定日数過去に遡って購入履歴のない顧客

# 2列目以降

軸説明
購入者＝購入＋未購入
購入＝再購入＋初購入
再購入1列目で表示された再購入者の中の再購入者
初購入1列目で表示された初購入者の中の再購入者
未購入1列目で表示された購入者の中の未購入者

# 合計行

合計行における初購入、再購入の判定は以下のルールで出力します。

初購入：画面中の商品のいずれかが初購入で、再購入がない。

再購入：画面中の商品のいずれかが再購入。

※合計行の顧客ID数＝各商品ID数の合計ではありません。

# 詳細トグル

詳細トグルをONにすることで購入を初購入と再購入に分けて表示できます。
