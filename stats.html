# 分析回数

※本マニュアルはユーザ権限が管理者の方々を対象にしています。

各メニューで行った分析回数を月ごとに集計したデータ(12か月分)を確認できます。

# メニュー別出力

上の表には、各メニューごとに集計された分析回数が表示されます。

# アカウント別出力

上の表中の各メニューをクリックすると、そのメニューにおけるアカウントごとのデータが表示されます。

# ファイルのダウンロード

表の左上に表示されているをクリックするとそれぞれの表をCSVファイルで出力し、ダウンロードできます。

# ユーザー・IPアドレス別CSV出力

年月を選択しユーザー・IPアドレス別CSV出力をクリックすることで、ユーザー・IPアドレス別CSVファイルをダウンロードできます。 フォーマットは以下の通りです。

  • 組織
  • ログインID
  • 氏名
  • IPアドレス
  • 各分析画面毎の分析回数
