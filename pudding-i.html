# 商品×顧客セグメント

商品／商品分類毎の顧客セグメント（ランク／PFV／年代）構成を集計します。商品／商品分類毎の顧客特性の違いを知る為に利用します。

# 条件設定画面

条件設定画面の共通操作はこちら

商品×顧客セグメントの条件設定の特徴的な部分について説明をします。

# 商品分類

商品×顧客セグメントの商品分類は分類、商品(分類指定／商品指定)から選択でき、選択したものによって分析の軸の出方が変わります。

# 分類

選択した分類階層の分類に従って分類ごとに結果が出力されます。

# 商品(分類指定／商品指定)

商品分類の共通操作はこちら

  • 分類指定

選択した分類に従って商品ごとに結果が出力されます。

  • 商品指定

選択した商品に従って、商品ごとに結果が出力されます。

商品指定の詳しい操作については以下のリンクを参照してください。

# オプション

商品×顧客セグメントのオプションは顧客セグメントとKPIの評価項目で構成されています。

# 評価表示

表示する値をどちらか選択します。

  • 構成比：各値を比率で表示します。

  • 実数：各値を実数で表示します。
