# 商品×KPI

このメニューでは、商品／分類毎にID-POSの任意の指標を集計します。

# 条件設定画面

# 商品分類

商品×KPIの商品分類は分類、商品(分類指定／商品指定)から選択でき、選択したものによって分析の軸の出方が変わります。

# 分類

選択した分類階層の分類に従って分類ごとに結果が出力されます。

# 商品(分類指定／商品指定)

商品分類の共通操作はこちら

  • 分類指定

選択した分類に従って商品ごとに結果が出力されます。

  • 商品指定

選択した商品に従って、商品ごとに結果が出力されます。
