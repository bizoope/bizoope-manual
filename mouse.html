# 匿名加工情報ダウンロード

このメニューでは、ユーザごとに作成したデータの情報表示・ダウンロードを行うことができます。

# リスト画面

リスト画面について説明します。

リスト画面はリストとステータスで構成されています。

# リスト

リストについて説明します。

① データを作成

クリックすると分析条件画面に飛びます。

② 更新

ページ(リスト)の更新を行います。

※ページ(リスト)は自動で更新されないので随時更新を行ってください。

③ 削除

リスト内のデータを選択し、削除ボタンをクリックすることでそのデータをリストから削除できます。

④ アイコン

アイコンは4種類あります。

# ステータス

リストからデータを選択すると分析条件が表示されます。

①ダウンロード

データをダウンロードします。

②条件を再利用

分析条件に飛び、選択したデータの分析条件を復元します。

③パスワード

解凍パスワードです。

# 分析条件画面

分析条件画面の説明をします。

この画面では、分析条件を指定します。

条件設定画面の共通操作はこちら

# 基本条件

基本条件について説明します。

  • 説明

入力した文がリストの説明欄に表示されます。

  • タイプ

データのタイプをID-POS/POSから選択できます。

ID-POSは会員の実績を、POSは会員・非会員(カードのない人)の実績を含みます。

ID-POSPOS
対象会員会員・非会員
集計項目年月日・時・分・店舗・商品・顧客年月日・店舗・商品
  • 追加ダウンロード

選択することで、商品マスタ・店舗マスタ・顧客属性マスタのファイルを併せてダウンロードできます。

詳細はダウンロードファイルへ

# 商品分類

※商品分類で選択できる商品・分類は契約時の分類のみです。

# 顧客

顧客について説明します。

  • 全顧客

全顧客を指定します。

  • 顧客指定

顧客ファイルから顧客を指定できます。

詳細はファイル読み込み仕様へ

  • 属性指定

データの作成に使用する顧客属性(年代・性別)を指定できます。

# ダウンロードファイル

ファイルはzipファイルにまとめてダウンロードします。

区切り文字はカンマ(,)、文字コードはShift-JISです。

パスワードを入力して解凍してください

# 商品マスタ

categories.csvは、カテゴリーコードをまとめたデータです。

# 店舗マスタ

追加ダウンロードで店舗マスタを選択すると、stores.csvのファイルを併せてダウンロードできます。

stores.csvは、店舗コード・店舗名・店舗の郵便番号をまとめたデータです。

# 顧客属性マスタ

追加ダウンロードで顧客属性マスタを選択すると、customers_attribute.csvのファイルを併せてダウンロードできます。

customers_attribute.csvは、顧客コード・年代・性別・顧客の住所(市区町村まで)をまとめたデータです。
