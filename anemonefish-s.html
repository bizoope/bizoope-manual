# 店舗・業態間送客

このメニューでは、対象店舗（群）-比較店舗（群）間の以下の２つの関係を示します。

1．対象-比較間の顧客分割（マーケット・セグメンテーション）

2．対象からみた相性分析（期間併買と同時併買）

# 条件設定画面

条件設定画面の共通操作はこちら

店舗・業態間送客の条件設定の特徴的な部分について説明をします。

# 店舗

分析に使用する、対象と比較の2つの店舗(群)の設定を行うことができます。

対象と比較はタブで切り替えて設定します。

対象で設定した店舗が比較の中に含まれている場合、分析時に自動で比較の中から除かれます。

# オプション

表示の絞り込み条件を設定できます。

併買率およびLift値が設定値以上のみ表示されます。

# 併買率、Lift値について

店舗・業態間送客において使用される2つの評価項目「併買率」「Lift値」について説明します。

# 併買率

自店舗群を利用したID数に対する、相手店舗との併用を行ったID数の比率です。 併用率が大きいほど、自店舗群を利用する顧客は相手店舗も利用する割合が高くなっています。

# Lift値

自店舗群の購買率に対する、併買率の倍数を示す指標です。 Lift値が大きいほど、自店舗群利用の際、相手店舗との併買が起こりやすくなっています
