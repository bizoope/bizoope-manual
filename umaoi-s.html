# 顧客動態統計（店舗スイッチ）

2期間中に、2つの店舗(群)間に発生した顧客の流入出を集計します。

# 条件設定画面

条件設定画面の共通操作はこちら

顧客動態統計(店舗スイッチ)の条件設定の特徴的な部分について説明をします。

# 期間

分析に使用する2期間の設定を行うことができます。

# 店舗

分析に使用する、対象と比較の2つの店舗(群)の設定を行うことができます。

対象と比較はタブで切り替えて設定します。

対象で設定した店舗が比較の中に含まれている場合、分析時に自動で比較の中から除かれます。

# 出力結果画面

出力結果画面の共通操作はこちら

顧客動態統計(店舗スイッチ)を二種類の表（集計表/定義表）で出力します。

# 定義表

流入出と継続顧客の内訳です。

# 動態セグメント列

動態セグメント列の〇／✕は、購買が有り／無しです。

# 流入出列

表示説明
流入前期が✕、当期が〇
継続前期、当期ともに〇
流出前期が〇、当期が✕
非購入前期、当期ともに✕

# Switch

Switch説明
対象へ比較から流出、対象へ流入
比較へ対象から流出、比較へ流入

Switch顧客の把握と、どの店舗からSwitchしたかを確認できます。

Switch列の〇をクリックすると、店舗一覧を表示できます。

# 集計表

顧客の流入出と継続を把握できます。

# ID数の各値

値説明
顧客計顧客動態統計の総合計
購入顧客計顧客計－非購入顧客計
非購入顧客計非購入の合計
当期顧客計当期が〇の合計
前期顧客計前期が〇の合計
流入計流入の合計
継続計継続の合計
流出計流出の合計

# 詳細トグル

詳細トグルをONにすることで、「Switch」「その他から」「その他へ」を表示できます。

# ID数の各値(詳細トグルON)

値説明
流入計
Switch対象が流入、比較が流入
その他からSwitch以外の流入
流出計
Switch対象が流出、比較が流入または継続
その他へSwitch以外の流出
