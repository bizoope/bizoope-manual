# ユーザ管理 HOW TO❓

※本マニュアルは、ユーザ権限が管理者の方々を対象にしています。

# ユーザ管理画面への遷移したい

  1. サービス一覧画面より右上にあるアイコンをクリックします

# ユーザを登録したい

  1. ユーザリストより左下にある＋アイコンをクリックします

# ユーザを更新したい

  1. ユーザリストの右側にある編集アイコンをクリックします

# ユーザを削除したい

  1. ユーザリストの右側にある[・・・]アイコンをクリックして、削除をクリックします

# パスワードをリセットしたい

  1. ユーザリストの右側にある[・・・]アイコンをクリックして、パスワードリセットをクリックします

  2. リセットをすると初期パスワードにパスワードが表示されます。このパスワードをユーザに通知してください。初期パスワードが空白になっているものは初期パスワードからユーザによりパスワードが変更されていることを表していまます。

# POS

  • POSユーザの設定をする場合は、下部にあるPOS開示サイトタブをクリックします

# POSユーザを登録したい

  1. 利用サービスのPOS分析をチェックしてください。

# 商品をハイライトするメーカー・取引先を設定したい

  1. [メーカーを設定する]または[取引先を設定する]をチェックして、対象となるメーカー・取引先を[あいまい検索]または[コード入力]より選択してください。

# 表示する商品はメーカー・取引先に限定したい

  1. [POS分析で閲覧可能商品は自社商品のみ]をチェックしてください。

# ツリーに表示するカテゴリを制限したい

  1. 分類指定をチェックして、対象の分類階層・分類をチェックしてください。

# ID-POS

  • ID-POSユーザの設定をする場合は、下部にあるPOS開示サイトタブをクリックします

# ID-POSユーザを登録したい

  1. 利用サービスのID-POS分析をチェックしてください。
