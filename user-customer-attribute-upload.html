# 顧客属性アップロード

※本マニュアルは、ユーザ権限が管理者の方々を対象にしています。

顧客明細xKPIで使用する顧客属性データのアップロードを行います。

# 操作方法

  1. ファイルをアップロードするボタン押下後、顧客属性ファイルを指定し名前を指定してください。その後、SAVEボタンを押すことでアップロードが開始します。既に登録されている名前は登録できません。
  2. 一覧のゴミ箱ボタンを押下することで削除ができます。

# 顧客属性ファイルレイアウト

  • 形式：csvファイル
  • ヘッダ行
    • 1項目には顧客コードを設定してください。
    • 2項目以降は属性名になります。顧客属性は最大10個まで指定できますが、10個未満でも構いません。
  • 明細行
    • 顧客コードとヘッダに対応する属性を設定してください。
    • 同顧客コードが存在している場合、属性はスペースで連結されて表示されます。例) 使用 未使用

例

顧客コード,クーポン,アプリ会員,キャンペーン
000000001,使用,◯
000000002,未使用,◯
000000003,使用,✕
000000004,,参加
