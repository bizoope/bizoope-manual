# 商品ユニーク顧客ベスト

このメニューでは、商品／分類毎にユニーク顧客のベストを集計します。 最も利用ID数の多い商品／分類を1位、1位のIDを除いて最も利用ID数の多い商品／分類を2位に、1位と2位のIDを除いた最も利用ID数の多い商品／分類を3位にと繰り返すベスト分析です

# 商品分類

商品_ユニーク顧客ベストの商品分類は分類、商品(分類指定／商品指定)から選択でき、選択したものによって分析の軸の出方が変わります。

# 分類

選択した分類階層の分類に従って分類ごとに結果が出力されます。

# 商品(分類指定／商品指定)

商品分類の共通操作はこちら

  • 分類指定

選択した分類に従って商品ごとに結果が出力されます。

  • 商品指定

選択した商品に従って、商品ごとに結果が出力されます。

# 1位の計算

1位は一番ID数が多いデータになります。この場合「パン」が1位となります。

# 2位以下の計算

パンを購買した顧客を除外して、ID数を再計算します 2位は「おにぎり」になります。
